nerdy.js

A data analysis and machine learning toolkit for Javascript.

Utility functions
  • parse a string of delimited data
  • evaluate a polynomial at some point
  • sum values in array
  • sum of squares values in array
  • mean of array
  • variance of array
  • standard deviation of array
  • minimum value of array
  • maximum value of array
  • median of array
  • skewness of array
  • kurtosis of array
Primary dataset model is nerdy.Dataset, supporting:
  • loading data from arrays or delimited text
  • querying the data
  • extracting arrays of numbers
  • update dataset
Available math tools:
  • function minimization
  • simulating random distributions
  • various vector distance metrics
Available stats tools:
  • descriptive statistics
  • one-way ANOVA, variance, covariance, and correlation methods
  • normalcdf, normalarea, and random.normal for dealing with normal distributions
  • linear regression (can be expanded to polynomial)
  • logistic regression (can be expanded to polynomial)
Available machine learning algorithms:
  • hierarchical clustering
  • k means clustering
  • k nearest neighbor
  • naive bayes
  • neural networks
  • principle component analysis
  • support vector machines
